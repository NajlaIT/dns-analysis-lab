# dns-analysis-lab
## Overview
This project focuses on analyzing DNS traffic to understand how domain names are resolved into IP addresses during real network activity.

## Tools Used
- Wireshark

## Methodology
- Captured live network traffic
- Visited multiple websites (Google, YouTube, LinkedIn, Wikipedia)
- Filtered DNS traffic using Wireshark filters
- Analyzed domain name queries

## Key Findings
- DNS queries are generated before establishing connections
- Major services like Google and YouTube generate frequent DNS requests
- Background services also create continuous DNS traffic

## Skills Learned
- DNS resolution process
- Network traffic filtering
- Packet analysis fundamentals
