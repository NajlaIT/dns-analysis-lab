# DNS Analysis Lab using Wireshark

## Overview
This project analyzes DNS traffic captured from a home Wi-Fi network using Wireshark. The goal is to understand how domain names are translated into IP addresses during real-time internet activity.

## Tools Used
- Wireshark

## Methodology
- Captured live network traffic from a Wi-Fi connection
- Visited multiple websites including Google, YouTube, and LinkedIn
- Applied DNS filters in Wireshark
- Inspected DNS query and response packets

## Key Findings
- DNS queries occur before establishing website connections
- Popular services such as Google and YouTube generate frequent DNS traffic
- Background applications also continuously generate DNS requests
- DNS plays a critical role in Internet communication

## Skills Demonstrated
- DNS traffic analysis
- Packet filtering using Wireshark
- Understanding domain name resolution
- Basic network traffic inspection
